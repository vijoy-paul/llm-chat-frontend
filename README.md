## Setup & Run


## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

The frontend expects the backend to be running at `http://localhost:3001` (or update the API URL in `src/components/Chatbot.jsx`).
